First steps: knowing the charts

The graphs of the ChIP-Seq, DNaseI-Seq and Methylation data are spline+ribbon-based scatter plots that represent the genomic coordinates on the x-axis and the averaged (solid line), minimum and maximum (shadow area) -log10(p-value), z-values or methylation levels for the ChIP-Seq, DNA-Seq and WGBS experiments on the y-axis, respectively.

  1. The title and subtitle show what it is being shown, and the filtering conditions.
  2. Transcripts, exons, UTRs, CDSs and TSSs (start and stop codons) are shown as special series in charts whose X axis represents chromosomical coordinates.
  3. The X axis title shows the genomic range of the search, plus the flanking window.
  4. Each chart can be (1) maximized or restored; (2) hidden; (3) the chart legend can be shown / hidden; (4) switch view described in (T) between the condensed gene layout (for each gene, only the transcript from the principal isoform is shown) and the complete one (all the transcripts are shown); (5) data supporting visible series in the chart can be browsed and saved.
  5. The legend shows items for each data series and genomic features, which can be hidden / shown by clicking on them.
  6. On any chart, a warning sign appears when one or more visible data series have no data. Clicking on it the interface shows the names of the related cellular types or diseases.
  7. A brief description of the type of chart is shown putting the pointer over this icon.
  8. Any chart with this symbol can be saved as a graphics file (PNG, JPG, PDF, SVG), as well as the data series used to generate the chart.

RNA-Seq data is represented in box-plots as an expression value for the whole gene or for each transcript (Fragments Per Kilobase Million).

  1. The title and subtitle show what it is being shown, and the filtering conditions.
  2. Boxplot, as well as its supporting points, are drawn. Boxplots numeric data which is shown when the user puts the cursor over it are the mean, average, Q1, Q3, minimum, maximum, and further outliers.
  3. The X axis title shows the genomic range of the search, plus the flanking window.
  4. Each chart can be (1) maximized or restored; (2) hidden; (3) the chart legend can be shown / hidden; (5) data supporting visible series in the chart can be browsed and saved.
  5. The legend shows items for each data series and genomic features, which can be hidden / shown by clicking on them.
  6. On any chart, a warning sign appears when one or more visible data series have no data. Clicking on it the interface shows the names of the related cellular types or diseases.
  7. A brief description of the type of chart is shown putting the pointer over this icon.
  8. Tools provided by plotly are available on each boxplot.

RNA-Seq data is also represented by heatmaps used to show Welch's pairwise t-test comparisons (at p-value level) between pairs of cellular types on the same gene or transcript.

  1. The title and subtitle show what it is being shown, and the filtering conditions.
  2. As the pairwise comparisons are done at gene or transcript level, there is a selector to navigate among them.
  3. The X axis title shows the genomic range of the search, plus the flanking window
  4. Each chart can be (1) maximized or restored; (2) hidden; (5) data supporting visible series in the chart can be browsed and saved.
  5. Hovering over any cell from the heatmap provides the names of the compared cellular types, and the obtained p-value. When any of the series only has a single element, the cell is drawn in black, with the text 'NA' (not available).
  6. On any chart, a warning sign appears when one or more visible data series have no data. Clicking on it the interface shows the names of the related cellular types or diseases.
  7. A brief description of the type of chart is shown putting the pointer over this icon.
  8. Any chart with this symbol can be saved as a graphics file (PNG, JPG, PDF, SVG), as well as the data series used to generate the chart.
