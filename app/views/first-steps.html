First steps: knowing the interface

  1. Reproducible searches, through query URLs.
  2. Basic BLUEPRINT data release metrics.
  3. Search field with autocomplete capabilities, based on EnsEMBL, GENCODE and REACTOME dumps, which understands union (+) and difference (-) set operators.
  4. Default flanking window around the query regions obtained from the search.
  5. Any chart with this symbol can be saved as a graphics file (PNG, JPG, PDF, SVG), as well as the data series used to generate the chart.
  6. Through the menu the user can access the analysis methods used on BLUEPRINT primary data, several help pages, and contact the developers

  1. Queries (genes, transcripts, complexes, pathways, ...) can return one or more query regions. Each query region is shown as a tab.
  2. Main features found in the query region are enumerated, providing links to their authoritative web sites.
  3. Metadata (donors, samples, analysis, ...) related to each query region is available to be browsed and saved.
  4. On the second step, the user can choose which cellular types with analysis results (i.e. the ones with filled circles) desires to initially inspect.
  5. Selected cellular types are marked in bold.
  6. It is possible to get from the cellular type the list of samples with analysis data for this search.

  1. On the third step, the user can choose which ones of the available charts for the selected cellular types with analysis results are initially shown.
  2. The list of selectable charts depend both on available results for the query and the selected cellular types in the previous step.
  3. It is also possible to initially filter the used analyses by the diagnosed disease related to the samples used in the experiments.

  1. There are several views which allow exploring the analysis data from different points of view. As of BDAP version 0.9.6, the available views are "General", "By tissue" and "Diseases by cellular type".
  2. Each cellular type with data on the query region is listed. Their data series can be switched on and off at once on all the charts. For each cellular type it is possible to see the names of the samples with data on the query region.
  3. It is possible to focus on a specific disease’s analysis data in “General” and “By tissue” views.
  4. Spline+ribbon-based scatter plots (1) are used for almost every chart type, such as the regulatory regions, methylation data and histone marks, showing the averaged series, minimum and maximum. Box-plot charts (2) are used for gene and transcript expression.
  5. Transcripts, exons, UTRs, CDSs and TSSs (start and stop codons) are shown as special series in charts whose x axis represents chromosomical coordinates.
  6. Each chart can be (1) maximized or restored; (2) hidden; (3) the chart legend can be shown / hidden; (4) switch view described in (T) between the condensed gene layout (for each gene, only the transcript from the principal isoform is shown) and the complete one (all the transcripts are shown); (5) data supporting visible series in the chart can be browsed and saved.
  7. On any chart, a warning sign appears when one or more visible data series have no data. Clicking on it the interface shows the names of the related cellular types or diseases.
  8. A brief description of the type of chart is shown putting the pointer over this icon
