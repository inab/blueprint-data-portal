First steps: knowing the interface

  1. Reproducible searches, through query URLs.
  2. Basic BLUEPRINT data release metrics.
  3. Search field with autocomplete capabilities, based on EnsEMBL, GENCODE and REACTOME dumps, which understands union (+) and difference (-) set operators.
  4. Default flanking window around the query regions obtained from the search.
  5. Any chart can be saved as a graphics file (PNG, JPG, PDF, SVG), as well as the data series used to generate the chart.
  6. Queries (genes, transcripts, pathways, ...) can return one or more query regions. Each query region is shown as a tab.
  7. Features found in the query region are enumerated, providing links to their authoritative web sites.
  8. Metadata (donors, samples, analysis, ...) related to each query region is available to be browsed and saved.
  9. On the second step, the user can enable which cellular types with analysis results desires to watch.

  1. There are several views which allow exploring the analysis data from different points of view. As of BDAP version 0.9.1, the available views are general, by tissue, and diseases by cellular type.
  2. Each cellular type with data on the query region is listed. Their data series on all the charts can be switched on and off at once. For each cellular type is possible to see the names of the samples with data on the query region.
  3. Segment-based scatter plots are used for almost each chart type, like regulatory regions, methylation data and histone marks. Boxplot charts are used for gene and transcript expression.
  4. Start and stop codons, gene regions, and gene start and stop features are shown on step charts.
  5. Each chart can be (1) maximized or restored; (2) hidden; (3) chart legend can be shown / hidden; (4) data supporting visible series in the chart can be browsed and saved.
